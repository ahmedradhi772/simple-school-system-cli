# نظام إدارة الطلاب البسيط عبر سطر الأوامر (Simple School System CLI)

##  نظرة عامة على المشروع

هذا المشروع هو تطبيق بسيط يعمل عبر سطر الأوامر (CLI)،
 تم تطويره لإدارة بيانات الطلاب، ويشمل عمليات الإضافة،
  الحذف، التعديل، وعرض معلومات الطلاب المسجلين والدروس الخاصة بهم.

##  التقنيات والمكتبات المستخدمة

تم بناء هذا المشروع بالاعتماد على الأدوات والمكتبات التالية:

| **Node.js** | بيئة التشغيل الأساسية. |
| **sqlite3** | مكتبة Node.js الرسمية للتفاعل مع قاعدة بيانات SQLite. |
| **readline-sync** | مكتبة لمعالجة المدخلات بشكل متزامن عبر سطر الأوامر (CLI). |

##  كيفية الإعداد والتشغيل (Getting Started)

لتشغيل المشروع على جهازك، يجب أن يكون لديك Node.js مثبتاً. اتبع الخطوات التالية بالترتيب:

### 1. استنساخ المستودع (Clone the Repository)

قم بنسخ المستودع إلى جهازك باستخدام أمر Git التالي:

git clone https://github.com/ahmedradhi772/simple-school-system-cli.git

### 2. تثبيت الاعتماديات (Install Dependencies)

يُنفذ هذا الأمر لتثبيت مكتبتي sqlite3 و readline-sync في مشروعك باستخدام مدير الحزم npm:
npm install sqlite3 readline-sync

### 3. تشغيل البرنامج
node app.js
